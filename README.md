# person_msgs
  * topic通信とサービスの実装に使えるファイルで構成されたROS2パッケージです。
## Person.msg
  * string型の"name"とuint8型の"age"を送信する際に使えるファイルです。
## Query2.srv
  * uint8型の"birthmoon"を渡し、string型の"birthstone"を返すサービスに使えるファイルです。
# インストール
  * 当パッケージはROS2環境を前提として今う
